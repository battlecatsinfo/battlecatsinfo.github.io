篩選表達式、排序表達式

如1 + 1、2 * 3等，是用來篩選和排序貓咪的運算式，類似數學的表達式，在程式語言裡通常稱做 expression。

  • 用在篩選上 篩選表達式將會算出一個結果，如果結果成立(true)，則會將該結果移除；如果是不成立(false)，則結果將會保留
    例如 range >= 400 篩選距離大於 400 的貓咪 例如 (dps > 40000) && (price <= 5000) 篩選 DPS 大於 40000 且價格小於 5000 的貓咪

  • 用在排序上 排序表達式將會算出一個數值，會以這個數值將篩選過的結果做由大到小的排序
    如果你沒有指定排序表達式，將會以貓咪的ID大小作為預設排序
    例如，輸入 dps 將會將dps由大到小排序
    例如，dps / cd 就是每秒的 DPS 產量（ADPS）
    例如，dps * range 就是 DPS 和距離的乘積
    如果看數值最小的排序，可以加上負號，如 -dps 就是從 DPS 最小排到最大

常數

有些預先定義的符號，稱為常數

如 RED 代表紅色敵人

如 AB_WAVE 代表波動能力

這些符號不用特別去記憶，點擊篩選按鈕就可以看到

運算子

類似數學中的運算式，可以用來作數值的運算和邏輯運算

運算子 解釋 最先結合順序(最高1)
* 乘法 1
/ 除法 1
% 取餘數 1
+ 加法 2
- 減法 2
> 大於 3
>= 大於等於 3
< 小於 3
<= 小於等於 3
== 等於 4
!= 不等於 4
& 位元AND 4
| 位元OR 4
^ 位元XOR 4
&& 邏輯AND 5
|| 邏輯OR 5

位元AND、位元OR、位元XOR平常較少用到，如果要用AND功能，請用&&，如果要用OR功能，請用||

另外 ( 和 ) 可以指定優先運算，如 -(1+2)*3 是 -9

另外-atk是負的意思，相當於乘-1，!atk是否(not)的意思，如果atk是0傳回1否則傳回0

  • max: 最大值，如max(1, 2, 3) = 3
  • min: 最小值，如min(1, 2, 3) = 1
  • sqrt: 平方根值，如sqrt(81) = 9
  • pow: pow(2, 4) = 16
  • round: 四捨五入，如round(0.5) = 1
  • random: 生成0-1之間的亂數（包含0，不包含1）
  • floor: 無條件捨去法，如floor(0.9) = 0
  • ceil: 無條件進位法，如ceil(0.1) = 1
  • sign: 如果輸入大於 0，傳回 1，如果輸入小於 0，傳回 -1，否則傳回 0 或 -0
